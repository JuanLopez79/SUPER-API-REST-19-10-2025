### 1. Inicializar repositorio:

```bash

```

### 2. Agregar Carpetas y archivos

```bash

git add .

```

### 3. Agregar descripcion del cambio

```bash

git commit -m "Descripcion del cambio"

```

### 4. Vincular a la nube de Github

```bash

git remote add origin https://github.com/github_username/repository_name.git

```

### 5. Seleccionar rama principal

```bash

git branch -M main

```

### 6. Reflejar cambios

```bash

git push -u origin main

```
